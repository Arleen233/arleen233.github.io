如何用hugo搭建个人博客

如何用Hugo搭建个人博客 安装Hugo（去hugo release页面下载hugo_xxx_windows64bit.zip）； 解压，将hugo.exe放到自己放软件的位置，并将hugo/加到PATH； hugo version 查看版本是否安装成功； 进入Hugo官网，点击quick start快速开始； step2（在自己喜欢的文件夹下面执行这一条命令：hugo new site [github用户名.github.io-creator]，会创建一个以[github用户名.github.io-creator]为名字的文件夹）； step3 执行： cd [github用户名.github.io-creator] //切换到该目录下 git init //创建git本地仓库 git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke //添加博客主题 echo ‘theme = “ananke”’ >> config.toml step4 hugo new posts/my-first-post.md //在posts目录下创建我的第一个博客（md文件），将draft属性修改为 false（不是草稿模式）； Step 5 hugo server -D //开启server访问地址可以看到自己的博客内容啦 Step 6 打开config.toml文件修改为languageCode = “zh-Hans”（简体中文） 在GitHub上面部署public，让别人可以访问网址看到这个网站，创建仓库，仓库名为[github用户名.github.io] 进入public目录下执行 git add . 执行 git commit -m 远程关联仓库 git remote add … git push -u origin master 设置GitHub pages然后就可以通过访问网址访问博客了。
